# Awesome Voice Input Methods 中文版

这是一个收集语音输入法、AI 听写工具、语音转文字应用以及相关开源项目的清单。

项目目标很简单：帮你快速找到那些可以“少打字、多说话”的工具，用在写作、聊天、办公、编程和日常记录里。

English version: [README.md](./README.md)
